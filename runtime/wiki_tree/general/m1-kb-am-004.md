# CLI Commands

> author: m1 · confidence: 0.85


- `/caveman lite|full|ultra` - switch caveman mode intensity
- `/caveman` - activate caveman mode
- `/exit` - exit session
- `claudefast -p` - probe/judge for verification
- `ssh -fN paperclipmini` - remote tunnel
- `TMUX= tmux ...` - nested tmux without $TMUX inheritance
