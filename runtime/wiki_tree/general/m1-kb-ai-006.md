# 5. Actionable Insights

> author: m1 · confidence: 0.85


### 5.1 For Sprint Contract Writing

1. **Done Criteria must map 1:1 to Verification Plan steps** — Every criterion needs a corresponding verification command
2. **Out of Scope must explicitly list exclusions** — Generator should not guess what's NOT included
3. **Risks table must include mitigations** — Risks without mitigations are not actionable
4. **Testability > Completeness** — A smaller testable scope beats a larger untestable scope
5. **Evaluator verdict format**: APPROVED (can start) o
